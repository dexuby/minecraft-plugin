## minecraft-plugin

[![](https://img.shields.io/jetbrains/plugin/v/24574.svg)](https://plugins.jetbrains.com/plugin/24574-minecraft-project-creator/)

Simple and easy to use IntelliJ IDEA plugin to create Minecraft server plugin projects written in Java.

### Target: Java 17

Supported languages:
- Java

Supported build systems:
- Maven

Supported server api versions:
- All (Auto updates)
